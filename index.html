What is Mesh?

Mesh is a token on the Vite network that uses an SBP node to reward its holders, all funds gained in the presale go to funding an SBP and all rewards from the SBP go to Mesh holders. MeshSBP is the only SBP to distribute 100% of it's rewards!

Services

Mesh SBP

The Mesh SBP we run allows us to distribute rewards to all the people who have staked the Mesh token. Mesh distributes 100% of it's rewards to stakers, this allows almost 50% ROI in one year!

Interface

Mesh provides you with a clean interface to interact with your staked coins. This way, you always know what to do, and how to do it.

Decentralization

Mesh runs on completely decentralized and trustless smart contracts giving you the peace of mind while using our services.

ViteX

Mesh will be listed on ViteX a DEX built into the vite network that is completely trustless and safe! ViteX is used by thousands of people across the globe!

Communications

We pledge to always have open communications and to be transparent so that you can have a streamlined experience.
